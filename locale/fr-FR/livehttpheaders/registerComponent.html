Afin que les informations des en-têtes soient disponibles dans l'onglet PageInfo de Mozilla/Firebird, il est nécessaire d'enregistrer le composant nsHeaderInfo. Ce composant se trouve dans le fichier Mozilla/Components/nsHeaderInfo.js.

Normalement, ceci est fait automatiquement lors de l'installation de LiveHTTPHeaders. Cependant, il arrive que l'enregistrement ne fonctionne pas. Voici donc la procédure à suivre afin de forcer Mozilla/Firebird à trouver et enregistrer le composant.

  • Supprimer le fichier Mozilla/Components/compreg.dat
  • Supprimer le fichier Mozilla/Components/xpti.dat
Ces deux fichiers devraient être recréés automatiquement lors du prochain démarrage de Mozilla/Firebird.

Note:Si vous avez installé LiveHTTPHeaders dans votre profile, vous devez d'abord copier le fichier nsHeaderInfo.js qui se trouve dans profile/components dans Mozilla/components avant de suivre les directives ci-dessus.
